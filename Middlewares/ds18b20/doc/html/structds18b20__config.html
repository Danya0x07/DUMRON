Подробное описание

Структура настроек датчика.

Разрешение значения температуры влияет на длительность измерения. Температурные границы позволяют установить диапазон температур, значения вне которого будут считаться как опасные, и датчик, результат последнего измерения с которого оказался вне этого диапазона будет отвечать на команду аварийного поиска Alarm Search.
