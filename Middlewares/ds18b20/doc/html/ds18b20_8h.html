Подробное описание

Драйвер температурных датчиков DS18B20.

Автор
Danya0x07

На текущий момент поддерживается работа только с одной линией 1-Wire.

Необходимо сделать:

Реализовать проверку контрольной суммы.

Реализовать динамический поиск датчиков на линии.

Реализовать поиск датчиков по тревоге (Alarm search).

◆ ds18b20_check_presense()

int8_t ds18b20_check_presense ( void  )

Отправляет импульс перезагрузки по линии 1-Wire.

Предполагается, что данная функция будет использоваться в качестве проверки до вызова остальных API-функций обращения с датчиками.

Возвращает
DS18B20_OK, если был получен импульс присутствия от датчиков,
DS18B20_ABSENSE, если не был получен импульс присутствия.

◆ ds18b20_get_result()

int8_t ds18b20_get_result ( const uint8_t *  address,
int32_t *  result 
)

Считывает результат измерения температуры.

Внимание
Эту функцию можно вызывать не ранее чем через соответствующее выбранному разрешению температуры время после вызова ds18b20_start_measurement(address).
Аргументы
[in]address8-байтовый массив с адресом опрашиваемого датчика или NULL, если на линии один единственный датчик.
[out]resultУказатель на переменную, в которую будет записан результат измерения.

Результат записывается в градусах Цельсия в "сыром" виде, т.е. 4 младших десятичных разряда отведены под дробную часть. Например, для 10.0625 результат будет 100625, а для -52.5 результат будет -525000.

Возвращает
DS18B20_OK, если всё прошло успешно,
DS18B20_ABSENSE, если если не был получен импульс присутствия,
DS18B20_CRC_MISMATCH если не совпали контрольные суммы.

◆ ds18b20_parse_result()

void ds18b20_parse_result ( int32_t  result,
int8_t *  integer,
uint16_t *  fractional 
)

Парсит результат измерения на целую и дробную части.

Вообще, распарсить результат измерения температуры может понадобиться в том случае, если его нужно отобразить на каком-нибудь дисплее, либо если нужна именно целая его часть. Если нужно сравнить результаты измерений, то делать это гораздо удобнее в "сыром" виде.

Аргументы
[in]resultРезультат измерения, полученный вызовом функции ds18b20_get_result(address, &result)
[out]integerУказатель на переменную, в которую будет записана целая часть результата *(с учётом знака)*, или NULL, если она не нужна.
[out]fractionalУказатель на переменную, в которую будет записана дробная часть результата *(без учёта знака)*, или NULL, если она не нужна.
См. также
ds18b20_get_result(const uint8_t *address, int32_t *result)
