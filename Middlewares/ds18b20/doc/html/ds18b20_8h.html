Подробное описание

Простая кроссплатформенная библиотека для датчиков температуры DS18B20.

На текущий момент поддерживается работа только с одной линией 1-Wire. Для расчёта контрольной суммы CRC реализован только побитовый алгоритм, поскольку выигрыш по скорости от табличного метода не будет иметь большого по сравнению с проигрышем по памяти значения вследствие невысокой скорости самого протокола 1-Wire.

Отправляет импульс перезагрузки по линии 1-Wire.

Предполагается, что данная функция будет использоваться в качестве проверки до вызова остальных API-функций обращения с датчиками.

Возвращает

DS18B20_EABSENT, если не был получен импульс присутствия.

◆ ds18b20_get_result()

int ds18b20_get_result ( const uint8_t *  address,
int32_t *  result 
)

Считывает результат измерения температуры.

Внимание
Эту функцию можно вызывать не ранее чем через соответствующее выбранному разрешению температуры время после вызова ds18b20_start_measurement().
Аргументы
[in]address8-байтовый массив с адресом опрашиваемого датчика или NULL, если на линии один единственный датчик.
[out]resultУказатель на переменную, в которую будет записан результат измерения.
Заметки
Результат записывается в градусах Цельсия в "сыром" виде, т.е. 4 младших десятичных разряда отведены под дробную часть. Например, для 10.0625 результат будет 100625, а для -52.5 результат будет -525000.
Возвращает
DS18B20_OK, если всё прошло успешно,
DS18B20_EABSENT, если если не был получен импульс присутствия,
DS18B20_ECRC, если не совпали контрольные суммы.

int ds18b20_measure ( const uint8_t *  address,
int32_t *  result 
)

Производит измерение температуры.

Внимание
Эта функция блокирует поток управления на время измерений.
Аргументы
[in]address8-байтовый массив с адресом опрашиваемого датчика или NULL, если на линии один единственный датчик.
[out]resultУказатель на переменную, в которую будет записан результат измерения.
Заметки
Формат результата измерения такой же, как у ds18b20_get_result().
Возвращает
DS18B20_OK, если всё прошло успешно,
DS18B20_EABSENT, если если не был получен импульс присутствия,
DS18B20_ECRC, если не совпали контрольные суммы.
DS18B20_EBUSY, если истёк срок ожидания (скорее всего означает короткое замыкание линии 1-Wire на землю).

◆ ds18b20_parse_result()

void ds18b20_parse_result ( int32_t  result,
int8_t *  integer,
uint16_t *  fractional 
)

Парсит результат измерения на целую и дробную части.

Вообще, распарсить результат измерения температуры может понадобиться в том случае, если его нужно отобразить на каком-нибудь дисплее, либо если нужна именно целая его часть. Если нужно сравнить результаты измерений, то делать это гораздо удобнее в "сыром" виде.

Аргументы
[in]resultРезультат измерения, полученный вызовом функции ds18b20_get_result() или ds18b20_measure().
[out]integerУказатель на переменную, в которую будет записана целая часть результата *(с учётом знака)*, или NULL, если она не нужна.
[out]fractionalУказатель на переменную, в которую будет записана дробная часть результата *(без учёта знака)*, или NULL, если она не нужна.
