◆ nrf24l01_crc_mode

enum nrf24l01_crc_mode

Настройки проверки трансивером контрольной суммы при получении пакета.

Контрольная сумма может либо не проверяться, либо быть размером в 1 байт, либо в 2 байта.

Заметки
Этот параметр у передатчика и приёмника должен быть одинаковым.
Внимание
Если используется автоподтверждение хотя бы на одном соединении, проверка контрольной суммы не может не использоваться.

◆ nrf24l01_pipe_mode

enum nrf24l01_pipe_mode

Режим входящего соединения приёмника.

Элементы перечислений
NRF24L01_PIPE_MODE_NOACK 

Не высылать автоподтверждения по этому соединению, размер принимаемой по этому соединению полезной нагрузки постоянен.

NRF24L01_PIPE_MODE_ACK 

Высылать автоподтверждения по этому соединению, размер принимаемой по этому соединению полезной нагрузки постоянен.

Заметки
Если по этому соединению высылаются автоподтверждения с полезной нагрузкой, то, даже если размер принимаемых по нему данных постоянен, нужно использовать режим NRF24L01_PIPE_MODE_ACK_DPL.
NRF24L01_PIPE_MODE_ACK_DPL 

Высылать автоподтверждения по этому соединению, размер принимаемой по этому соединению полезной нагрузки непостоянен.

◆ nrf24l01_pipe_number

enum nrf24l01_pipe_number

Номер входящего соединения.

6 входящих соединений позволяют приёмнику принимать пакеты от 6-ти передатчиков, работающих на одинаковой частоте с ним.
