◆ nrf24l01_measure_noise()

void nrf24l01_measure_noise ( uint8_t *  snapshot_buff,
uint8_t  start_ch,
uint8_t  end_ch 
)

Делает "снимок" зашумлённости частотного диапазона.

15 раз проходит по диапазону каналов, на каждом вызывая nrf24l01_detect_signal(), соответственно, для каждого канала накапливается значение от 0 до 0xF, показывающее его зашумлённость.

Аргументы
[out]snapshot_buffМассив, в который будут записаны значения зашумлённости для каждого канала. Должен быть размером не менее end_ch - start_ch + 1.
start_chНачальный канал снимка. Должен быть в диапазоне от 0 до NRF_CHANNELS - 1.
end_chКонечный канал снимка. Должен быть в диапазоне от 0 до NRF_CHANNELS - 1.
Предупреждения
start_ch <= end_ch.

◆ nrf24l01_start_output_carrier()

void nrf24l01_start_output_carrier ( enum nrf24l01_power  power,
uint8_t  channel 
)

Заставляет трансивер транслировать свою несущую волну.

Предупреждения
Данная функция должна использоваться только для тестирования трансивера на предмет бракованности, она не совместима со стандартными режимами работы и при вызове перезаписывает настройки на специальные для трансляции несущей. Однако до неё можно вызвать nrf24l01_set_rf_channel(), чтобы установить частотный канал, на котором будет трансляция.
Аргументы
powerМощность сигнала, с которой нужно транслировать.
channelРадиочастотный канал, на котором нужно транслировать.
