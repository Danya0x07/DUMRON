Структура с настройками входящего соединения.

Заметки
Подразумевается, что настройка соединений будет производиться после инициализации трансивера, т.е. после создания и заполнения структуры nrf24l01_rx_config и передачи её аргументом в функцию nrf24l01_rx_configure().

См. определение в файле nrf24l01.h строка 334

Поля

◆ address

union { ... } address

Адрес соединения. Для соединений 0 и 1 используется указатель array, который должен указывать на массив, размер которого определяется атрибутом nrf24l01_rx_config.addr_size, установленным ранее вызовом nrf24l01_rx_configure(). Для соединений 2-5 используется только младший байт - lsb. (Старшие байты наследуются от адреса соединения 1).

◆ array

uint8_t* array

для соединений 0 и 1

См. определение в файле nrf24l01.h строка 344

◆ lsb

uint8_t lsb

для соединений 2-5

См. определение в файле nrf24l01.h строка 345

◆ pld_size

uint8_t pld_size

Размер пакетов, принимаемых по этому соединению (от 1 до 32 байт). Это поле не имеет значения, если выбран режим соединения NRF24L01_PIPE_MODE_ACK_DPL.
