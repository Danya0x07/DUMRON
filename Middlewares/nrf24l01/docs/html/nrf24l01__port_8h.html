◆ _spi_transfer_byte()

static uint8_t _spi_transfer_byte ( uint8_t  byte)
inlinestatic

Обменивается байтом по SPI.

Заметки
Если используемый стандарт языка не поддерживает inline, можно использовать макрос.
Ошибка:
Если функция обмена байтом по SPI работает с volatile-данными (регистрами МК) и находится в отдельной единице трансляции, то для arm-none-eabi-gcc v6.3.1 имеет место баг, вызванный оптимизацией компилятора. Баг проявляется на уровнях оптимизации, отличных от -O0 и -Og. Симптомы: некорректные значения при передаче/приёме данных по SPI.
