◆ nrf24l01_read_pld_size()

int nrf24l01_read_pld_size ( void  )

Считывает размер полезной нагрузки, находящейся первой в очереди приёма.

Имеет смысл, если используется динамическая длина полезной нагрузки.

Заметки
Имеет смысл как для приёмника, так и для передатчика: чтобы узнать размер полезной нагрузки, принятой вместе с пакетом автоподтверждения.
Возвращает
Размер полезной нагрузки (от 1 до 32 байт),
0, если очередь приёма пуста.
