◆ nrf24l01_rx_configure_minimal()

int nrf24l01_rx_configure_minimal ( uint8_t  pld_size)

Инициализирует трансивер как приёмник настройками по-умолчанию.

Минимальная конфигурация приёмника и передатчика позволяет протестировать их работоспособность. Она подразумевает односторонний обмен данными с подтверждениями и фиксированным размером полезной нагрузки.

Заметки
Эта функция сама настраивает входное соединение 0 для приёма данных от передатчика. Все остальные настройки трансивера остаются в значениях по умолчанию, указанных в даташите.
Внимание
Данная функция будет корректно работать только если настройки трансивера ранее не менялись (со времени подачи на него питания).
Аргументы
pld_sizeРазмер полезной нагрузки (от 1 до 32 байт). Передатчик должен соблюдать такой же.
Возвращает
0, если инициализация прошла успешно,
-1, в противном случае (скорее всего плохой контакт).
