◆ nrf24l01_rx_configure_default()

int nrf24l01_rx_configure_default ( void  )

Инициализирует трансивер как приёмник настройками по-умолчанию.

Минимальная конфигурация приёмника и передатчика позволяет протестировать их работоспособность, используя самые базовые их возможности: односторонний обмен данными без подтверждений и повторных отправок.

Заметки
Эта функция сама настраивает входное соединение 0 для приёма данных от передатчика. Все остальные настройки трансивера остаются в значениях по умолчанию, указанных в даташите.
Внимание
Данная функция будет корректно работать только если настройки трансивера ранее не менялись (со времени подачи на него питания).
Возвращает
NRF24L01_OK, если инициализация прошла успешно,
NRF24L01_ERR_BUS, в противном случае.
