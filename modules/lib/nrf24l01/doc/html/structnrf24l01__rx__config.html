◆ en_irq

uint8_t en_irq

Включённые прерывания. Является маской из NRF24L01_IRQ. Определяет, в случае происхождения каких событий логический уровень на пине IRQ трансивера будет сбасываться в 0.

◆ features

uint8_t features

Используемые возможности трансивера, относящиеся к расширенным. Является маской из NRF24L01_FEATURES.

◆ rf_channel

uint8_t rf_channel

Радиочастотный канал. Должен находиться в пределе от 0 до NRF24L01_CHANNELS - 1. Должен совпадать с каналом передатчика.
