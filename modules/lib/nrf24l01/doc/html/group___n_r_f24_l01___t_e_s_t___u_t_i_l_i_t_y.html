Функции трансивера, позволяющие анализировать работу радиосвязи и его самого.

uint8_t  start_ch,

Делает "снимок" зашумлённости частотного диапазона.

15 раз проходит по диапазону каналов, на каждом вызывая nrf24l01_detect_signal(), соответственно, для каждого канала накапливается значение от 0 до 0xF, показывающее его зашумлённость.

Аргументы
[out]snapshot_buffМассив, в который будут записаны значения зашумлённости для каждого канала. Должен быть размером не менее end_ch - start_ch + 1.
start_chНачальный канал снимка. Должен быть в диапазоне от 0 до NRF_CHANNELS - 1.
end_chКонечный канал снимка. Должен быть в диапазоне от 0 до NRF_CHANNELS - 1.
Предупреждения
start_ch должен быть < end_ch.
